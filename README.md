# Sales Forecasting using ARIMA and Prophet

## Project Overview
This project predicts future sales using Time Series Forecasting techniques.

## Objectives
- Analyze historical sales trends
- Detect seasonality patterns
- Compare ARIMA and Prophet models
- Evaluate performance using MAE and RMSE

## Technologies Used
- Python
- Pandas
- Matplotlib
- ARIMA
- Prophet
- Scikit-learn

## Results
Prophet achieved lower forecasting error compared to ARIMA on the test dataset.
